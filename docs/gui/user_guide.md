# EG-HG Desktop Application User Guide

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python -m gui.main`

## Features

### Graph Canvas
- Interactive visualization of existential graphs
- Zoom and pan with mouse
- Select and manipulate graph elements

### Bullpen Tools
- Create blank sheets
- Apply logical templates
- Parse and generate CLIF text
